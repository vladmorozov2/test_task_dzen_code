Опис лабораторної роботи

У файлі lab_data.csv зібрано результати фізичного експерименту. Ці дані складають із позицій та швидкостей об'єктів. Для зібраних даних треба продовжити аналіз ("EDA", Exploratory data analysis).

а) Проведіть аналіз в системі jupyter notebook. (Бібліотеки на вибір: matplotlib, seaborn, pyplot, cv2)
- 1. Оцініть щільність абсолютної швидкості тіл різними методами: Гаусова суміш, ОМВ (нормальна), оцінка ядрами. Візуалізуйте їх (2d, 3d)
- 2. Зробить те ж саме для абсолютного прискорення прискорення. Додатково побудуйте емпіричну функцію розподілу
- 3. Формулою Надараї-Ватсона згладьте швидкість і прискорення об'єктів (асолютні значення) (X - позиції, Y - швидкість або прискорення)
- 4. Повторити 3, але методом локальної регерсії.

'r1x', 'r1y', - позиції об'єкта 1

'v2x', 'v2y', - швидкості об'єкта 2
